# Tech Interview Tasks

Take-home assignments for technical interviews at Cyclomedia.

| Task | Description |
|------|-------------|
| [Data Engineer](DataEngineerTask.md) | Stream and process a COPC point cloud, save structured output |
| [Product Engineer](ProductEngineerTask.md) | Build a collaborative point cloud viewer with real-time camera sync |

Open the task file for full requirements and evaluation criteria.

## Submission

Create your own **public** repository on GitHub (or another version control hub), commit your solution there, and send us the link. We want to see your commit history and how the work evolved.
